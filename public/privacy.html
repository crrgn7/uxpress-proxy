Privacy Policy

UXPress Proxy — Last updated 15 June 2026

This policy explains what data is handled by the UXPress Proxy service at uxpress-proxy.vercel.app and how it is used. The UXPress WordPress plugin sends data through this proxy to provide AI-assisted UX review features.

1. Proxy registration

Admin-triggered

When a WordPress site administrator connects their account through the Freemius opt-in flow, the UXPress plugin sends a one-time registration request to this proxy containing:

  • The WordPress site URL (normalized to origin only)
  • A Freemius install ID (used to verify the site is a legitimate UXPress installation)

The proxy verifies the install ID against the Freemius API using developer credentials stored in Vercel environment variables. On success it returns an HMAC-signed authentication token that is stored in the WordPress database. No registration occurs before the Freemius opt-in is completed.

2. Deep Review (AI analysis)

Editor-triggered

When a logged-in editor explicitly clicks Run AI Review in the Gutenberg sidebar, the plugin sends the following data through this proxy to Google Gemini:

  • Page text content and block structure of the currently open page
  • A screenshot of the Gutenberg editor canvas
  • The site authentication token (to verify the request is from a registered site)
  • Relevant page metadata (page type, score context)

UXPress does not intentionally store Deep Review payloads on this proxy after the request completes. The proxy relays the request to Google Gemini and returns the response.

Data processed by Google Gemini is subject to Google's applicable terms and privacy policies: Google Cloud Privacy Notice — Google Cloud Terms of Service.

Please do not submit pages containing sensitive, confidential, or regulated personal data (health records, financial information, personal identifiers of third parties, etc.) to Deep Review.

3. What is not collected

  • No payment card data
  • No form field values or form submission contents
  • No visitor personal data (names, emails, IP addresses of your site's visitors)
  • No cookies are set by this proxy

4. Visitor friction and performance signals

On Pro plans, the UXPress plugin may collect aggregate visitor friction signals (Core Web Vitals, rage clicks, scroll depth, form abandonment flags) directly to your WordPress installation via the REST API. These signals are stored locally in your WordPress database and are not transmitted through this proxy or to any external service.

5. Freemius (licensing and accounts)

UXPress uses Freemius for software licensing, subscription management, checkout, account connection, plan entitlement checks, and plugin updates. Freemius operates independently of this proxy. See Freemius Privacy Policy and Freemius Terms of Service.

6. Infrastructure

This proxy is hosted on Vercel (Vercel Inc., USA). Request logs may be retained by Vercel for operational purposes per Vercel's Privacy Policy. The proxy itself does not write request payloads to any persistent storage.

7. Contact

For privacy questions or data requests: uxpress.support@gmail.com
